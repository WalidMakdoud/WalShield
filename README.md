# 🛡️ WalShield

WalShield is a network security monitoring toolkit developed in Python with a modern React dashboard. It helps detect common network attacks and visualize security events through an easy-to-use web interface.

---

## 📌 Features

- 🔍 ARP Network Scanner
- 🚨 Port Scanner Detection
- 🌐 DoS Attack Detection
- 📡 Wi-Fi Deauthentication Attack Detection
- 📊 Web Dashboard
- 💾 SQLite Database for Logs
- ⚡ FastAPI Backend API
- 🎨 React + Vite Frontend

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/WalShield.git
cd WalShield
```

---

### 2. Create a virtual environment

Linux/macOS

```bash
python3 -m venv myenv
source myenv/bin/activate
```

Windows

```cmd
python -m venv myenv
myenv\Scripts\activate
```

---

### 3. Install Python dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Install frontend dependencies

```bash
cd frontend
npm install
```

---

## 🚀 Running the Project

### Start the FastAPI Backend

```bash
cd BackEnd
uvicorn main:app --reload
```

The backend runs on:

```
http://127.0.0.1:8000
```

API documentation:

```
http://127.0.0.1:8000/docs
```

---

### Start the React Frontend

```bash
cd frontend
npm run dev
```

The frontend runs on:

```
http://localhost:5173
```

---

### Run the Security Agents

Open another terminal and execute the desired detector.

ARP Scanner

```bash
python Agent/arp_scn.py
```

Port Scanner Detection

```bash
python Agent/port_scaner_detection.py
```

DoS Detection

```bash
python Agent/Dos_Scanner.py
```

Deauthentication Detection

```bash
python Agent/Deauth_Scanner.py
```
