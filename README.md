## рџљЂ Running the Project

### вљЎ Quick Start (Automated)

**Linux/macOS**
```bash
chmod a+x run.sh stop.sh
sudo ./run.sh
```

**Windows** *(Run as Administrator)*
```bat
start.bat
```

These scripts automatically start the FastAPI backend, the React frontend, and all security agents in one command.  
To stop everything gracefully on Linux/macOS:

```bash
./stop.sh
```

---

### рџ”§ Manual Start

#### Start the FastAPI Backend

```bash
cd BackEnd
uvicorn main:app --reload
```

Backend runs on: `http://127.0.0.1:8000`  
API docs: `http://127.0.0.1:8000/docs`

#### Start the React Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on: `http://localhost:5173`

#### Run the Security Agents

Open separate terminals for each agent:

```bash
# ARP Scanner
python Agent/arp_scn.py

# Port Scanner Detection
python Agent/port_scaner_detection.py

# DoS Detection
python Agent/Dos_Scanner.py

# Deauthentication Detection
python Agent/Deauth_Scanner.py
```

> вљ пёЏ Most agents require **root/administrator privileges** due to raw packet capture (Scapy).
